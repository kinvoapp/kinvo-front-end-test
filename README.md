### Setting up Dev

Here's a brief intro about what a developer must do in order to start developing
the project further:

```shell
git clone https://github.com/matheusm/kinvo.git
cd kinvo/
yarn install

# add .env file
touch .env

# add VITE_API_URL in .env

yarn dev
```
